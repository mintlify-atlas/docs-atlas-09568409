Creates a new service in the database with the provided information.

## Endpoint

```
POST /api/services
```

## Request

### Headers

```
Content-Type: application/json
```

### Body Parameters

<ParamField body="imagen" type="string" required>
  URL of the service image. Must be provided and cannot be empty after trimming whitespace.
</ParamField>

<ParamField body="titulo" type="string" required>
  Service title or plan name. Required field with a maximum length of 120 characters. Whitespace is automatically trimmed.
</ParamField>

<ParamField body="precio" type="number" required>
  Service price. Must be a number greater than or equal to 0. Negative values are not allowed.
</ParamField>

<ParamField body="descuento" type="number">
  Discount percentage. Must be between 0 and 100. Defaults to 0 if not provided.
</ParamField>

<ParamField body="descripcion" type="string" required>
  Service description detailing benefits and features. Required field with a maximum length of 500 characters. Whitespace is automatically trimmed.
</ParamField>

### cURL Example

```bash
curl -X POST http://localhost:5000/api/services \
  -H "Content-Type: application/json" \
  -d '{
    "imagen": "https://ejemplo.com/premium.jpg",
    "titulo": "Plan Premium",
    "precio": 9.99,
    "descuento": 15,
    "descripcion": "Acceso completo a todos los canales HD y contenido exclusivo"
  }'
```

### JavaScript Example

```javascript
const newService = {
  imagen: "https://ejemplo.com/premium.jpg",
  titulo: "Plan Premium",
  precio: 9.99,
  descuento: 15,
  descripcion: "Acceso completo a todos los canales HD y contenido exclusivo"
};

const response = await fetch('http://localhost:5000/api/services', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(newService)
});

const createdService = await response.json();
```

## Implementation

Controller function at `/home/daytona/workspace/source/astro_backend/src/controllers/serviceController.js:30-37`:

```javascript
export const createService = async (req, res) => {
  try {
    const service = await Service.create(req.body);
    res.status(201).json(service);
  } catch (error) {
    res.status(400).json({ message: 'Error al crear el servicio', error: error.message });
  }
};
```

## Validation Rules

Schema validation is enforced from `/home/daytona/workspace/source/astro_backend/src/models/Service.js:3-37`:

| Field | Validation Rules |
|-------|------------------|
| `imagen` | Required, trimmed |
| `titulo` | Required, trimmed, max 120 characters |
| `precio` | Required, minimum value 0 |
| `descuento` | Optional, default 0, min 0, max 100 |
| `descripcion` | Required, trimmed, max 500 characters |

## Common Validation Errors

| Error | Cause |
|-------|-------|
| `La URL de la imagen es obligatoria` | Missing or empty `imagen` field |
| `El título del servicio es obligatorio` | Missing or empty `titulo` field |
| `El título no puede exceder 120 caracteres` | `titulo` exceeds maximum length |
| `El precio es obligatorio` | Missing `precio` field |
| `El precio no puede ser negativo` | `precio` is less than 0 |
| `El descuento no puede ser negativo` | `descuento` is less than 0 |
| `El descuento no puede superar el 100%` | `descuento` is greater than 100 |
| `La descripción es obligatoria` | Missing or empty `descripcion` field |
| `La descripción no puede exceder 500 caracteres` | `descripcion` exceeds maximum length |

## Notes

- The `createdAt` and `updatedAt` timestamps are automatically generated by MongoDB
- The `_id` field is auto-generated and does not need to be provided
- All string fields are automatically trimmed of leading/trailing whitespace
- The `descuento` field defaults to 0 if not provided

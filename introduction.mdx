# Welcome to Astro Services

Astro Services is a modern full-stack web application designed for showcasing and managing streaming and IPTV services. Built with cutting-edge technologies, it provides a complete solution for displaying services in a dynamic grid with real-time pricing, discount calculations, and comprehensive service management.

## What is Astro Services?

Astro Services combines a React-based frontend with an Express.js backend and MongoDB database to create a seamless platform for streaming service marketplaces. The application features:

- **Dynamic Service Grid**: Display streaming services with images, pricing, and discount information
- **Full CRUD API**: Complete REST API for managing services
- **Real-time Updates**: Service data is synchronized across the frontend and backend
- **Docker-Ready**: Containerized setup for easy deployment
- **Modern Stack**: React 19, Vite 7, Tailwind CSS 4, Express 4, MongoDB

## Service Data Model

Each service in Astro Services includes:

- **Image**: URL to service logo or promotional image
- **Title**: Service name (max 120 characters)
- **Price**: Base price in your preferred currency
- **Discount**: Percentage discount (0-100%)
- **Description**: Service benefits and features (max 500 characters)

<Note>
All service data is stored in MongoDB with automatic timestamp tracking and validation.
</Note>
